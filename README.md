# Anki-MessierObjects
Scrapes www.messier.seds.org and creates anki flash cards for each messier object

## Cards created

### Constellation
What constellation is the object in?

### Type
What is the type of the object?

### Name
What is the name of the object?

## Requirements
* genanki
* BeautifulSoup4
* requests
